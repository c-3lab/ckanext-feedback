# Like Module

Allows users to "like" registered data resources, and enables viewing the "number of likes" for each data resource.

## Benefits of Implementation

* Improved User Engagement
  * Users can easily provide feedback on data resources, increasing user engagement.
  
* Understanding Data Resource Quality
  * By viewing the popularity of data resources numerically, it becomes possible to prioritize improvements and maintenance for highly useful data resources.
  * Visualizing popular data resources makes it easier for other users to become interested, promoting data utilization.

## Feature Description

* By pressing the like button at the top right of the screen, the icon color and number change, allowing you to evaluate the data resource.

  **[Like Module - Evaluation Action]**
  ![Like Module - Evaluation Action Image](../assets/likes_image.jpg)

* The following two items can be visualized:
  * Number of evaluations for each data resource
  * Total number of evaluations for data resources within a package
